#Leiloes_uc11

#Sistema de Leilão
#Descrição
#Este projeto é um sistema simples de leilão, desenvolvido em Java, que 
#permite o cadastro de produtos e seus respectivos valores iniciais. Os produtos cadastrados são armazenados em uma
#tabela, onde podem ser gerenciados e vendidos utilizando o ID do produto.

#Funcionalidades
#Cadastro de Produtos: Permite cadastrar novos produtos com um nome e um valor inicial.
#Listagem de Produtos: Exibe uma tabela com todos os produtos cadastrados, incluindo seus IDs, nomes e valores.
#Venda de Produtos: Permite a venda de um produto específico utilizando seu ID, removendo-o da lista de produtos disponíveis.
#Atualização de Produto: Possibilidade de alterar informações do produto, como nome e valor, antes da venda.

#Tecnologias Utilizadas
#Java: Linguagem principal usada para desenvolvimento do sistema.
#Banco de Dados MYSQl: Um banco de dados relacional é utilizado para armazenar as informações dos produtos.
